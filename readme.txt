◆ kakoi.exe

nokakoiをグリッド表示にしてリアクションできるようにしたものです。

その他はnokakoiと同じです。
https://github.com/betonetojp/nokakoi

ESCキー：設定画面
F9キー ：コンテンツの折り返し表示の切り替え（余白ダブルクリックでも動作）
F10キー：ユーザーミュートとキーワード通知の設定画面（余白右クリックでも動作）
F11キー：メイン画面の表示と非表示
F12キー：ポストバーの表示と非表示（F1キーでも動作）

1 ～ 0：リアクションを選択
W / カーソルキー[↑]：イベント選択上移動
S / カーソルキー[↓]：イベント選択下移動
A / カーソルキー[←]：Webビューを開く（イベントを右クリックでも動作）
F / カーソルキー[→]：リアクションを送信（イベントをダブルクリックでも動作）
C：Webビューを閉じる
Ctrl + Shift + A：メインフォームをアクティブにするグローバルホットキー
